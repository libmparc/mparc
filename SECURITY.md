# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## What Constitutes a Vulnerability

- Normal security vulnerability
- Undefined behavior
- Memory leaks

## Reporting a Vulnerability

1. Go to issues
2. Make a new issue
3. Tag it with security
4. Fill the details
5. Publish the issue
